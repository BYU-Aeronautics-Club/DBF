Sub-team Specialties

Take a gander at some of the things our sub-teams do!

  • Everything
  • Leadership
  • Aerodynamics
  • Graphics
  • Structures
  • Propulsion
  • Manufacturing
  • Systems
  • Systems

  • Current Team Leads

    Get to know some of our current team leads.

  • Meetings

    Sometimes necessary, but usually not.

  • Communication

    If you don't want meetings, communicate.

  • Motivation

    We want to be motivators, not managers.

  • Airfoil Analysis

    Using XFoil to analyse and select airfoils.

  • Wing Analysis

    Using XFLR5 and other tools to design wings and tails.

  • Stability Analysis

    Using XFLR5 and other tools to analyse stability of wings and tails.

  • CFD

    Computational Fluid Dynamic analysis using STARCCM+ and other tools.

  • Wind Tunnel Testing

    Using both big and small wind tunnels as needed.

  • Part Modeling

    Using CAD for parametric models.

  • Part Assemblies

    Using CAD to simulate part interactions.

  • Drawings/Renderings

    To show off our sweet designs.

  • FEA

    Finite Element Analysis using ANSYS and other tools for wing and fuselage structures.

  • Flutter

    Working with Aero team to analyse aerodynamic flutter.

  • Structural Testing

    Yield, Fatigue, Failure, etc.

  • Propellers

    Analysing and choosing propellers for each flight mission.

  • Motors

    Analysing and choosing a motor.

  • Electronics

    Choosing the right speed controller, batteries, etc.

  • Wind Tunnel Testing

    Analysing propulsion system performance in the wind tunnels.

  • Prototyping

    Working with all the subteams to make prototypes for testing

  • Composites

    Working with composites to build super nice parts.

  • CNC

    Using Computer Numerical Control to hotwire wing cores,mill molds and prototypes, etc.

  • 3D Printing

    Use additive manufacturing for various parts and prototypes.

  • Laser Cutting

    Use laser cutting for lots of prototypes and parts.

  • System Design

    Work with all the subteams to make sure all the pieces fit together.

  • Specialty Systems

    Work on the wild card requirements for the year (e.g. bomb drops).

  • Simulator

    Put the design in the simulator for an additional check and practice.

  • Flight Testing

    Over and over and over.

  • Fly-off

    Don't mess it up...
